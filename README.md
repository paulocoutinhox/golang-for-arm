# Compile Go for ARM

It compile the **Go** form ARM host.  

## Instructions

> git clone https://github.com/prsolucoes/golang-for-arm.git    
> cd golang-for-arm  
> ./build.sh  

*Obs: You need docker to use it*

# Test

After build with success, you can execute:

> ./run.sh  

 *Obs: You need docker to use it*

# Issues
